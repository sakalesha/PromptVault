# PromptVault Project Report
Date: March 21, 2026

## 1. Project Overview
PromptVault is a sophisticated, full-stack web application designed for managing, optimizing, and collaborating on AI prompts. It provides a centralized repository for individuals and teams to store, version, and analyze their prompt engineering efforts.

## 2. Key Features
- **Centralized Prompt Management**: Organize prompts with categories, tags, and search functionality.
- **AI-Powered Prompt Generation**: Integrated with Google Gemini (via @google/genai) to generate high-quality prompts from simple descriptions.
- **Version History & Diffing**: Track changes over time with a detailed history and visual diff view to compare versions.
- **Team Collaboration**: Robust team management system allowing users to create teams, invite members, and collaborate on shared prompt libraries.
- **Analytics Dashboard**: Visualize prompt performance, usage statistics, and copy counts to identify top-performing assets.
- **Public Sharing**: Securely share individual prompts via public URLs with customizable access controls.
- **Templates**: Support for prompt templates with dynamic variables for reusable workflows.
- **Responsive Design**: Optimized for both desktop and mobile devices with a modern, dark-themed interface.

## 3. Technical Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4.0, Lucide React (icons), Motion (animations)
- **Backend/Database**: Firebase (Firestore, Authentication)
- **AI Integration**: Google Gemini API (@google/genai)
- **Utilities**: React Router (navigation), React Hot Toast (notifications), React Markdown (rendering), Diff (version comparison)

## 4. Component Architecture
The application follows a modular component-based architecture:
- **Dashboard**: The central hub managing state and layout.
- **Sidebar**: Navigation, team switching, and category/template filtering.
- **PromptEditor**: A comprehensive interface for creating and editing prompts with tag and category support.
- **PromptCard**: Displays prompt metadata, quick actions, and preview.
- **AnalyticsDashboard**: Renders usage data and performance metrics.
- **TeamManagement**: Handles member invitations, role management, and team settings.
- **PromptHistory**: Provides a timeline of changes with the ability to revert or compare versions.
